🔐 Cyber Security Mini Projects Suite

A collection of beginner-friendly cybersecurity tools demonstrating core concepts like encryption, logging, and password security.

🚀 Overview

This repository showcases four essential cybersecurity mini projects designed to provide hands-on understanding of basic security mechanisms such as encryption, data protection, and input monitoring.

🧩 Projects Included
🔤 Caesar Cipher (Text Encryption)

A classic encryption technique that shifts characters based on a user-defined key to secure text data.

🖼️ Image Encryption Tool

Implements basic image security using pixel manipulation:

Pixel value swapping
Mathematical transformations
⌨️ Keylogger (Educational)

Captures and logs keystrokes into a file to demonstrate how input monitoring works.

⚠️ Strictly for educational purposes only.

🔑 Password Strength Checker

Analyzes password strength based on:

Length
Uppercase & lowercase usage
Numbers
Special characters
